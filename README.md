📊 Sales Analysis Dashboard (Power BI Project)
📌 Overview

This project is an interactive Sales Analysis Dashboard built using Power BI to analyze sales performance, product trends, and customer insights.
It provides a complete business view through multiple dashboards with dynamic filtering and visual analytics.

🚀 Features
🔹 1. Sales Analysis Dashboard
Total Sales, Revenue, Orders, and Return Metrics
Sales trend over time (date-wise analysis)
Region-wise sales distribution (donut chart)
Manager-wise sales performance
Product-wise revenue contribution
Month-over-month growth tracking
Dynamic Quarter filter
Adjustable Sales Parameter (range filter)
